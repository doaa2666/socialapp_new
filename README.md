# Social Media App

A full-featured social media application built with Node.js, TypeScript, and a modular architecture. This app supports user authentication, posts, comments, friend requests, and email notifications.

---

## Features

- **User Management**: Registration, login, password hashing, JWT authentication.
- **Posts & Comments**: Create, read, update, delete posts and comments.
- **Friend Requests**: Send, accept, and reject friend requests.
- **Email Notifications**: Email verification and notifications.
- **Secure & Modular Architecture**: Clean folder structure with modular services, controllers, and repositories.

---

## Tech Stack

- **Backend**: Node.js, TypeScript, Express.js
